NETBREAKER v1.0 — CLASSIFIED REFERENCE DOCUMENT
§1 — Terminal Commands
missions
List all missions and their status.
start <id>
Begin a mission (e.g. start M01). Loads the mission filesystem.
ls
List files in the current directory.
ls -la
List all files including hidden ones (starting with a dot).
cd <directory>
Change directory. Use cd .. to go up one level.
cat <filename>
Read and display a file's contents.
grep <pattern> <filename>
Search for a pattern inside a file. Returns matching lines.
decode --base64 <string>
Decode a Base64-encoded string.
decode --hex <string>
Decode a hex-encoded string.
ping <ip>
Ping an IP address to check if it is online.
ps aux
List all running processes on the current system.
submit <answer>
Submit your answer for the current puzzle (e.g. submit PHANTOM).
clear
Clear the terminal output.
help / man
Open this manual.
whoami / uptime / ifconfig
System information commands.
§2 — Puzzle Types
BOTH File Investigation
Navigate the filesystem using ls, ls -la, cd, and cat. Hidden files start with a dot — only visible with ls -la.
BOTH Log Analysis
Read log files with cat or search them with grep. Look for repeated failures, unusual IPs, or suspicious timestamps.
GHOST Password Cracking
Find the hash file, then use common patterns — dictionary words, l33tspeak (a→4, e→3), appended numbers.
BOTH Decode Messages
Find encoded files in the filesystem, then use decode --base64 or decode --hex in the terminal to reveal the plaintext.
BOTH Binary Puzzles
Find a binary file with cat. Convert each 8-bit group to ASCII. 01000001 = 65 = A.
BOTH Network Mapping
Use ping on IP addresses found in config files. Find the vulnerable or unauthorized host.
BOTH Process Investigation
Run ps aux and examine the output carefully. Look for high CPU, suspicious names, or dot-prefixed processes.
BOTH Permissions Puzzle
Run ls -la to see file permissions. rwxrwxrwx = world-writable = dangerous. Submit the vulnerable file path.
BOTH Digital Breadcrumbs
Follow a chain of clues across multiple files. Each cat or grep leads to the next. The final destination is your answer.
BOTH Timed Puzzles
The clock starts when the puzzle loads. Find and decode the answer before time runs out.
