NETBREAKER v0.2.0 — CLASSIFIED REFERENCE DOCUMENT
§1 — Terminal Commands
missions
List all missions, their status, and required reputation.
start <id>
Begin a mission (e.g. start M01). Requires minimum reputation.
ls / ls -la [path]
List directory contents. -la reveals hidden dot files.
cd <dir>
Change directory. cd .. goes up one level. cd / goes to root.
cat <file>
Read a file's contents.
grep <pattern> <file>
Search for a pattern inside a file.
decode --base64 <str>
Decode a Base64 string.
decode --hex <str>
Decode a hex string.
ping <ip>
Ping an IP address.
ps aux
List running processes.
tools
List your unlocked tools and their usage.
submit <answer>
Submit your answer for the active puzzle.
pwd / whoami / clear / help
Utility commands.
§2 — Puzzle Types
BOTH File Investigation
Navigate with ls, cd, cat. Hidden files need ls -la.
BOTH Log Analysis
Read logs with cat, search with grep. Look for patterns.
BOTH Decode Messages
Find encoded files, use decode --base64 or decode --hex.
BOTH Network Mapping
Read scan results in the filesystem. Find open ports or unauthorized hosts.
BOTH Process Investigation
Run ps aux. Spot rogue or anomalous processes by PID.
BOTH Permissions Puzzle
ls -la shows permissions. rwxrwxrwx = world-writable = dangerous.
BOTH Binary Decode
Find binary file, convert 8-bit groups to ASCII. A=65=01000001.
BOTH Digital Breadcrumbs
Follow clues across multiple files. Each one leads to the next.
BOTH Timed Puzzles
Clock starts on load. Find and decode before time expires.
§3 — Reputation & Progression
Reputation is earned by solving puzzles. Lose rep for wrong answers.
Higher missions require a minimum reputation to unlock.
Act 1 — 0 rep required  |  Act 2 — 100 rep required  |  Act 3 — 250 rep required
