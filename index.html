List all missions and their status.

List directory contents. -la reveals hidden files.

Clear the terminal output.
§2 — Puzzle Types
BOTH File Investigation
Navigate a fake filesystem with ls and cat. Hidden files start with a dot (.). Use ls -la to reveal them. Look for unusual sizes, dates, or names.
BOTH Log Analysis
Read server logs to identify patterns. Ghost: erase incriminating entries. Sentinel: identify the intruder's IP from timestamps and failed auth patterns.
GHOST Password Cracking
Given a hash, find the original password. Try dictionary words, l33tspeak (a→4, e→3, o→0), and appended numbers. Use crack --hash to try the wordlist.
BOTH Decode Messages
Intercepted messages may be Base64, hex, Caesar cipher, or binary. Use decode commands or work it out manually.
BOTH Binary Puzzles
Convert binary to ASCII. Each group of 8 bits = 1 byte. 01000001 = 65 = 'A'. Read left to right, most significant bit first.
BOTH Network Mapping
Ghost: find vulnerable nodes (open ports). Sentinel: identify unauthorized hosts. Compare known vs. all active devices.
BOTH Digital Breadcrumbs
Follow a trail of clues across logs, files, and DNS records. Each clue leads to the next. Ghost: trace the target. Sentinel: trace the attacker.
BOTH Process Investigation
Use ps aux to list processes. Look for unusual names, high CPU, or hidden PIDs. Ghost: find the security monitor. Sentinel: find and kill the malicious process.
BOTH Permissions Puzzle
Unix permissions: rwxrwxrwx (owner/group/other). r=4, w=2, x=1. chmod 755 = rwxr-xr-x. Ghost: exploit over-permissioned files. Sentinel: lock them down.
BOTH Time-Based Puzzles
Solve under a countdown. Read before you act. Failed timed puzzles increase detection risk and reduce reputation.
BOTH Error Code Manual
Certain error codes reveal exploitable conditions or required patches. Cross-reference the §3 Error Database below.
